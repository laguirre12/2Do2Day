About

Touday(too-day) is a webpage that gives users advice on what to do for a particular event. The webpage also gives the user the freedom to add advice to the website for others to see.

For example, if the user is at work. They can click work, and some action would pop up on the screen. The user can either take that advice or click the refresh button that would show a different action.

The point of this webapp is so that people could try new or different things during the course of their regular schedule.

Touday is meant to help people get out of their comfort zones, and being that there can be many different actions we feel that a user should be able to add a suggestion. However, they must follow the Rules for posting suggestions:

  • No usage of profanities/curse words (whatever you want to call them).
  • The suggestions must be relevant to the specific category.
  • The suggestions should be meant to help people, and, thus, cannot have any hurtful comments targeting either a specific group or a user.
  • Special characters will not be allowed in posts. If a post has certain special characters it will not be added as a suggestion. (e.g. â€œ{ } < > * @, etc.)
